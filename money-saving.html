Money Saving App

Looking to make a big purchase? This App could help you get there

My Process

User Storys

As a user, I want to see a dashboard of my finances clearly and visually, so that I can see how much I am spending on what at a glance

As a user, I want to be able to view, filter, and interact with information on my finances in a variety of ways, so that I can access further details when I want to.

As a user, I need to be able to tell the tool what my savings goal is and how long I have to reach it, so that I can save accordingly

User Flow

Guided by the user story's I created 3 user flows that I felt addressed the needs of the users and was easy to navigate.

Wireframing

I then got to work on low fidelity wireframes. Using pen and paper, I sketched a layout, making sure to keep the user needs in mind.

From there I moved onto Figma and built up my designs in grey scale.

style-guide

Once I had a wireframe I was happy with, I moved on to the style. I went though a number of different colour and text combinations. I wanting to make sure the app was portraying the right feel.

Mock-Up

From there I tested my design and found the pain points. After some changes and more testing, I have a money saving app!
